## 当前状态诊断（对标哥飞标准）

### 做对了的
- 已有182个程序化SEO页面（z/目录 - 14主星×13宫位）
- 首页有内链指向z/页面（40个链接）
- FAQPage Schema已加

### 做错了/有风险的

- 首页内容略杂（紫微+工具站+API文档混在一起）→ 关键词密度稀释风险（对标哥飞ChatGPT4o.ai降权案例）

- 缺少"分门别类"的落实 —— z/页面的H2应更系统地罗列长尾关键词
- 首页缺少OG:image（分享到微信/社交平台没有图片）
- z/页面互相之间几乎没有内链（权重分散）
- 工具站和紫微站共用首页 → 主题不集中
- 英文软文没有cron自动发布（现在是手动或半自动）

## 方案一：SEO策略（分三个阶段）

### 阶段一：基础修复（本周完成）

#### 1.1 首页聚焦化
哥飞原文教训：**首页膨胀导致降权**（ChatGPT4o.ai首页从400词膨胀到两三千词 → 谷歌语义分析认为主题不集中 → 曝光骤降）。

行动：
- 把工具站（Text Tools Suite）从首页下移到独立的入口点
- 首页只留三个核心模块：排盘工具 + 14主星/12宫位导航 + 软文列表
- API文档入口保持但放在footer
- 首页核心关键词密度维持在合理范围

#### 1.2 z/页面noindex策略调整
哥飞原文：页面不被索引 = 没有SEO价值。但AdSense审核需要内容"安全"。

方案：**分层处理**
- 选择30-50个最高搜索量的z/页面去掉noindex（如紫微/天机/太阳/武曲等在命宫的页面）
- 其余保持noindex
- 等AdSense通过后全部去掉noindex

快速判断高价值z/页面：用Semrush查"紫微+主星+命宫"、"紫微+主星+财帛宫"等组合的搜索量。

#### 1.3 英文内容爆发
哥飞核心：**做英文站，不是只做英文，可以加上多语言支持，但首要目标是英文世界。** 英文ECPM $3-6，中文$0.1-0.3。

行动：
- 英文软文从3篇/周提高到每天1篇（7篇/周）
- 蹭关键词：Chinese astrology, Ba Zi, Four Pillars, Zi Wei Dou Shu, Chinese fortune telling, Bazi reading
- 每篇英文文章配1200×630的OG图片（用Pillow生成）
- 文章发布后提交IndexNow（Bing）加速收录

#### 1.4 z/页面互相内链
哥飞原文：**内链是SEO中最容易被忽视但效果显著的环节。**

行动：
- 每个z/页面末尾加"相关命盘"模块，链接到同主星不同宫位 + 同宫位不同主星
- 例如"紫微在命宮"页面 → 链接到"紫微在兄弟宮"、"天机在命宮"、"太阳在命宮"

### 阶段二：增长飞轮（2-4周后）

#### 2.1 内容型AI工具站飞轮
哥飞原创概念：**用户使用工具产生的内容自动展示在公开页面 → Google收录 → 更多流量 → 更多用户 → 更多内容。**

紫微斗数天然适合：
- 用户排盘后的命盘数据摘要 → 自动生成公开页面（如 /charts/20260519-14xx.html）
- 页面内容包含：命宫主星、四化分布、12宫位主星
- 每个页面都是独特的长尾关键词（"XX年XX月XX日XX时出生 紫微命盘"）
- 类似Sticker.Show的用户生成内容飞轮

具体实现：
1. 用户免费排盘后，系统自动保存一份匿名化数据（只留主星/宫位/四化，不留个人信息）
2. 自动生成 /charts/{date}-{hash}.html 页面
3. 新页面写入sitemap
4. 关键：用户越多 → 命盘页面越多 → Google收录越多 → 流量越多

#### 2.2 "分门别类、罗列成语"强化
哥飞六字真言。

当前已做：14主星×12宫位=168个页面。
可以扩展：
- 加上"14主星 + 四化"组合（如"紫微化权在命宮"）→ 14×4=56个新页面
- 加上"主星+辅星"组合（如"紫微+天府在命宮"）→ 约50个高频组合页面
- 加上"五行局"维度（金四局/木三局等在命宫的解读）
- 每个新页面都是独立的长尾关键词入口

#### 2.3 工具站独立域名
哥飞：**单域名单关键词策略。** 让工具站独立出去，避免稀释紫微站的首页主题。

行动：
- 工具站移到独立的textools.site（或类似域名）
- 紫微站首页回到纯紫微主题
- 两个站之间做友情链接（互相导流+传递权重）
- 每个站独立积累权重，互不干扰

### 阶段三：程序化SEO放大（1-2个月）

#### 3.1 命盘页面穷举
哥飞案例：**distance.to（129万页面/月）和DateTimeGo（1.5万页面/月，$3万广告费）都是穷举法。**

紫微的穷举方向：
- 所有主星组合×所有宫位 = 穷举百万级页面
- 每个出生日期一个页面（"2026年5月19日出生的紫微命盘"）
- 按年份、月份、性别、时辰的组合批量生成
- 但注意：**首页不要膨胀**，程序化页面放到独立子目录

#### 3.2 城市+命理页面
类似哥飞Phone Number Generator的国别策略：
- 每个城市一个页面："北京人紫微命盘特点"、"上海人紫微命盘分析"、"纽约华人紫微命盘解读"
- 按人口排序，先做一线城市，逐步覆盖

## 方案二：变现路径（哥飞路径：Adsense → 联盟 → 订阅 → 直接收费）

### 阶段一：Adsense变现（审核通过后立即生效）
哥飞原文：
- **收入公式**：月收入 = 30 × UV基数 × ECPM × PV/UV比 × 广告位数
- 命理/占星类ECPM：3-6美元（欧美用户）

当前状态：Adsense已提审（pub-5545263418745440），等待审核结果。

**广告位布局（等审核通过后操作）**：
参考12章广告优化：
1. 排盘结果上方放一个展示广告（用户等待结果的注意力集中区）
2. 命盘解读内容中间插一个（内容中间ECPM最高）
3. 页面底部放一个（不影响体验）
4. 软文页面：标题下方一个 + 内容中间一个 + 底部一个

### 阶段二：Freemium订阅（同步推进）

- **内容限制**：免费只给500字解读，完整解读付费
- **功能限制**：合盘、流年、择日Pro专属
- **去水印**：免费版解读末尾带"Powered by ZiweiAPI"，付费版去掉

### 阶段三：联盟导购（需一定流量）

#### 变现计算（按哥飞公式）

## 五、哥飞核心原则复读

- **养网站防老**：一次建站，持续变现10年。不要着急，紫微站是你的树苗
- **有比没有好**：先上线，再优化。不追求完美
- **英文优先**：同样流量，英文收入是中文30倍
- **首页不要膨胀**：保持主题集中，不要什么都放首页
- **内链>外链**：先做好内链，再做外链
- **内容型飞轮**：让用户帮你产生内容，你只做工具
