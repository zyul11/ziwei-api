紫微斗數入門指南

什麼是紫微斗數？

紫微斗數是中國傳統命理學中最具系統性與邏輯性的術數之一，起源於宋代，相傳由陳摶（希夷先生）所創。它以出生年月日時為基礎，透過複雜的排盤系統，推演出一個人的先天命運格局。

與八字（四柱推命）不同，紫微斗數以星曜分布為核心，將命盤分為十二宮位，每個宮位對應人生不同領域，如財運、事業、感情、健康等。

紫微斗數的基本元素

1. 十四主星

紫微斗數的核心是十四顆主星，分為六組：

星組星曜
紫微星系紫微、天機、太陽、武曲、天同
天府星系天府、太陰、貪狼、巨門、天相、天梁、七殺、破軍
南北斗輔星左輔、右弼、文昌、文曲、天魁、天鉞、祿存、擎羊、陀羅

其中 紫微星 為帝王星，代表尊貴、領導力；天機星 主智慧、謀略；太陽星 主光明、貴人；武曲星 主財富、剛毅。

2. 十二宮位

命盤分為十二宮位，逆時針排列：

  1. 命宮 — 性格、命運核心
  2. 兄弟宮 — 手足緣分、人際支援
  3. 夫妻宮 — 婚姻、感情伴侶
  4. 子女宮 — 子女緣、創造力
  5. 財帛宮 — 財運、金錢觀
  6. 疾厄宮 — 健康、疾病傾向
  7. 遷移宮 — 外出運、環境變動
  8. 交友宮 — 社交、朋友、合夥
  9. 官祿宮 — 事業、學業成就
  10. 田宅宮 — 房產、家庭根基
  11. 福德宮 — 精神生活、福氣
  12. 父母宮 — 父母緣、遺傳特質

3. 四化星

四化（化祿、化權、化科、化忌）是紫微斗數中動態變化的關鍵，它們賦予星曜不同的能量傾向：

  • 化祿 — 增加、豐盛、好運
  • 化權 — 權力、控制、責任
  • 化科 — 名聲、學問、貴人
  • 化忌 — 困擾、缺失、課題

如何開始學習？

步驟一：準備排盤工具

傳統上需要查萬年曆與手工排盤，但現代可以直接使用紫微斗數 API：

GET https://ziweiapi.site/api/ziwei/命盤

輸入出生年月日時（需知道農曆或透過 API 轉換），即可獲得完整的命盤資料。

步驟二：看懂命盤結構

命盤分為十二宮格，每個宮格內包含：

  • 該宮位的宮干地支（如「丙寅」、「丁卯」）
  • 落在此宮的主星與輔星
  • 如果有四化，會標註在星曜旁

步驟三：從命宮開始解讀

命宮是解盤的起點。觀察命宮內的主星，就能大致了解一個人核心性格：

  • 紫微在命宮：領袖氣質，自尊心強，有威嚴
  • 天機在命宮：聰明機敏，善於謀劃，變化多
  • 太陽在命宮：熱情開朗，樂於助人，光明磊落
  • 武曲在命宮：剛毅果斷，重視實際，財運佳

步驟四：串聯十二宮位

單一宮位只說明了部分情況，真正的解盤需要**三合方（三方四正）**的綜合判斷。例如命宮、財帛宮、官祿宮形成「命三方」，彼此互相影響。

常見入門誤區

「紫微斗數是宿命論？」

不是。紫微斗數揭示的是先天趨勢與性格傾向，而非不可改變的命運。了解命盤是為了「知命而後改運」，透過後天努力調整行為模式。

「星曜吉凶是絕對的？」

沒有絕對的吉星或凶星。例如七殺星被認為是殺氣重的星，但在軍警、外科醫生等行業中可以發揮極大優勢。星曜的吉凶取決於宮位組合、四化引動與大運流年。

下一步

這份指南只是紫微斗數世界的敲門磚。建議初學者：

  1. 先熟悉十四主星的基本特性
  2. 練習看自己的命盤十二宮
  3. 學習四化飛星的基本規則
  4. 從大運（十年運勢）開始練習推斷

使用 紫微斗數 API 可以快速取得命盤資料，讓你專注在解讀與學習上。
